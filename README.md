# 🚀 File Tools Pack - Web File Manager & Toolbox

**📌 Description**  
A sleek, fully functional, mobile-first web replica of the **File Tools Pack** Android app.  
Browse, organize, copy, delete, rename files & folders. Preview images, play videos, view documents — all inside your browser. Includes storage overview, gallery, recent files and toolbox features. Built with pure HTML5 + CSS3 + Bootstrap 5 + jQuery.

**✨ Features**  
- 📁 Full file explorer with folders & files  
- 🖼️ Image gallery with instant preview  
- 🎬 Video library with playable demo videos  
- 📄 Document viewer  
- 📊 Storage usage dashboard  
- 🔍 Smart search across all files  
- ✏️ Rename, delete, copy, move (context menu)  
- 📤 Fake upload & new folder creation  
- 📱 100% responsive & touch-friendly  
- 🌐 Works offline after first load  

**🏷️ Tags / Keywords**  
`file-manager-web`, `online-file-explorer`, `file-tools-pack`, `web-file-organizer`, `document-viewer`, `image-gallery-web`, `video-player-browser`, `storage-analyzer`, `file-rename-delete`, `responsive-file-manager`, `html5-file-tools`, `browser-file-manager`

**🛠️ How to run**  
1. Download all 4 files into the **same folder**  
2. Open `index.html` in any modern browser  
3. Start managing your demo files instantly!
