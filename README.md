# File-Tools-Pack---Web-File-Manager-Toolbox
A sleek, fully functional, mobile-first web replica of the **File Tools Pack** Android app.   Browse, organize, copy, delete, rename files &amp; folders. Preview images, play videos, view documents — all inside your browser. Includes storage overview, gallery, recent files and toolbox features. Built with pure HTML5 + CSS3 + Bootstrap 5 + jQuery.
